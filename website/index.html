01 / Features

Everything you need,
nothing you don't.

Built for simplicity and privacy. No bloat, no tracking, no accounts.

Audio flows directly between you.

Entavi uses WebRTC to route audio peer-to-peer — no media relay, no middleman server ever touches your voice. Once peers are connected, the call continues even if the signaling server goes offline.

Device A
DTLS-SRTP encrypted

Noise suppression

RNNoise processes your mic in real-time. Keyboards, fans, and background noise get filtered out before your voice leaves the device.

Host controls

Kick peers, force-mute, and password-protect rooms. Full control over who's in and what they can do.

Encrypted chat

Text messages travel over the same encrypted WebRTC data channel as audio. The signaling server never sees message content.

Web app + cross-platform

Download the native app for macOS, Windows, or Linux — or use web.entavi.app directly in your browser. No install needed.

02 / How it works

Three steps. That's it.

No sign-ups, no configuration. Getting started takes seconds.

01

Create a room

Open Entavi and hit "Start a room." You get a unique 6-character code instantly. Name it whatever you like.

02

Share the code

Send the room code to whoever you want to talk to — text it, post it, whatever. They enter it in the app or web version.

03

Talk directly

Once they join, audio flows directly between you. Encrypted, peer-to-peer, with no server in between. The signaling server could go offline and your call would continue uninterrupted.

Signaling server (Cloudflare Workers) — only helps peers find each other during setup. It never handles audio data and can be self-hosted. See the self-host guide below.

03 / Security

Private by design.

Your voice never passes through our servers. We couldn't listen even if we wanted to.

  • Audio and chat encrypted with DTLS-SRTP — standard WebRTC encryption protocol
  • No central server ever routes or stores your audio data
  • No accounts — zero personal information collected or required
  • No telemetry — the app doesn't phone home or track usage
  • Open source — the full codebase is public on GitHub for audit
"There is no Entavi server holding your audio. The audio never leaves the path between your devices."

04 / Self-Host

Deploy your own signaling server.

Run the signaling relay on your own Cloudflare account. The free tier is all you need.

01

Prerequisites

You'll need a Cloudflare account (free tier works), Node.js, and npm installed.

02

Clone and install

git clone https://github.com/avdoseferovic/entavi.git cd entavi/signaling-server && npm install
03

Configure

Edit wrangler.toml and set the name field to your preferred worker name.

04

Deploy

npx wrangler login npm run deploy
05

Connect the app

Copy your worker URL (e.g. wss://your-worker.workers.dev/ws), open Entavi, and paste it into the Signaling server field under Advanced settings.

05 / FAQ

Frequently asked questions.

Do I need to create an account?
No. Entavi requires no accounts, no registration, and no personal information. Just open the app, create or join a room, and talk.
Is it really peer-to-peer?
Yes. Audio flows directly between participants over WebRTC. The signaling server only helps peers find each other — it never touches your audio. Once connected, the server could go offline and your call would continue.
What platforms are supported?
Entavi is available for macOS (Apple Silicon), Windows (64-bit), and Linux (.deb and AppImage). You can also use it directly in your browser at web.entavi.app — no download required.
How many people can be in a room?
Rooms work best with small groups. Since audio is peer-to-peer, each participant connects directly to every other participant. Performance depends on your network, but small group conversations work great.
Is it encrypted?
Yes. All audio and chat messages are encrypted using DTLS-SRTP, the standard encryption built into WebRTC. Since data never passes through our servers, no one — not even us — can listen to your conversations or read your messages.
Is it free and open source?
Yes, completely free and open source. You can view the full source code, audit it, and contribute on GitHub.
